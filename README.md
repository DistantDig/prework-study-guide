# Prework Study Guide Webpage

## Description

A study guide for course pre-work. This study guide is a collection of notes for the UC Berkley Coding Bootcamp. At time of writting it contains a small collection of notes for HTML, CSS, Git, and Javascript. This project serves as an introduction to web design by being both a collection of sccumulated knowledge as well as a hands on practice project.

## Usage

This project holds information collected over the prework regarding different topics for web design. Each topic is listed along with a handfull of notes regarding their respective categories. Through Chrome DevTools, the console can also be viewed to see all of the topics listed through JavaScript. A random topic is selected at the end of this list as a recomendation for which topic to study next.
